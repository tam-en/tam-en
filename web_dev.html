Gentrification and Dislocation in Seattle

HTML, CSS, Javascript, React, Mapbox, Chart.js

The result of a 1.5-day hackathon project/collaboration with one other developer, two UX designers, and a data scientist. The goal was to use data to represent gentrification trends in Seattle. My main contributions were CSS implementation of the UX designers’ wireframes, media queries & alternative grid for mobile version, and tenant relocation ordinance chart with Chart.js.

GitHub repo: github.com/009Kings/seattle-gentrified

Live version: seattle-gentrified.herokuapp.com
