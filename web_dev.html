Here, Now (contentment log)

HTML, CSS, Python, Django

What: Application that lets users log how content they feel on a particular day and the what/where/when of moments of that brought them feelings of contentment, joy, or pleasure. Kind of an alternative to a gratitude journal, which for some of us translates into “journal of things I feel I should be more grateful for.” Instead, this app asks users to focus on what actually made them feel good in a particular moment, because those little things matter. This app is still a work in progress and has not been deployed yet, but I’m happy to demo it in person.

Github repo: http://blah blahblah

Gentrification and Dislocation in Seattle
(hackathon project)

HTML, CSS, Javascript, React, Mapbox, ChartsJS

What: The result a a 1.5-day hackathon project/collaboration with one other developer, two UX designers, and a data scientist. The goal was to use data to represent gentrification trends in Seattle. My main contributions were CSS implementation of UX designers wireframes and appropriate media queries for mobile version as well as implementation of ChartsJS representation of tenant relocation ordinance reports data.

Github repo: https://github.com/009Kings/seattle-gentrified

Deployed version: https://seattle-gentrified.herokuapp.com/
